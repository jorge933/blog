## Instalar o projeto na sua maquina

```bash
  git clone https://github.com/jorge933/blog.git
  cd blog
  npm install
  npm run init-db
```

crie um arquivo .env e o ajuste como o .env example
```bash
  npm start
```

Depois disso acesse http://localhost:5000/
